# Figure Reproduction

## Required behavior

- Do not modify raw results.
- Generate figures from manifest-traceable inputs.
- Save scripts and outputs in predictable locations.

## Steps

1. Identify the target figure and exact input result files.
2. Verify all inputs are linked from a result manifest.
3. Generate or update the figure script with minimal changes.
4. Save figures under `results/figures/` unless another path is requested.
5. Verify output file exists and labels/legends match the intended metrics.

## Output

Report input files, script path, figure path, and checks performed.
